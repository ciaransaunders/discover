# NewsApp

## Setup

1. Install Xcode 26+ from the App Store  
2. Check out the repository.
3. Open `Package.swift` in Xcode by double-clicking it, or from Xcode select **File > Open > Package.swift**.
4. Xcode will automatically resolve the package and build the targets.
5. Select your development team in Signing & Capabilities (Required for physical devices).
6. Select a simulator and press **⌘R**.

## Requirements
- macOS 15.0+
- Xcode 26+

## Running Tests
Run `make test` inside the `NewsApp-Swift` folder to safely compile and execute all unit tests.
